O que é GitHub Pages?

GitHub Pages é um serviço de hospedagem gratuito oferecido pelo GitHub que permite publicar sites estáticos diretamente de um repositório GitHub.

Saiba mais sobre GitHub Pages

Guia Rápido

  1. Crie uma conta no GitHub (ou faça login)
  2. Crie um novo repositório com o nome username.github.io (substitua "username" pelo seu nome de usuário do GitHub)
  3. Escolha um template abaixo ou crie seu próprio HTML
  4. Faça upload dos arquivos para o repositório
  5. Espere alguns minutos e acesse https://username.github.io

Como Publicar

  1. Vá para seu repositório username.github.io no GitHub
  2. Clique em "Add file" > "Create new file"
  3. Nomeie o arquivo como "index.html"
  4. Cole o código gerado
  5. Clique em "Commit new file" na parte inferior da página
  6. Acesse https://username.github.io em alguns minutos
